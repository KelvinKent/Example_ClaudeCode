# Example_ClaudeCode

Este repositório foi criado como exemplo de uso do **Claude Code** — a interface de linha de comando (CLI) oficial da Anthropic para o Claude.

## Sobre o Projeto

`Example_ClaudeCode` é um projeto de demonstração que ilustra como o Claude Code pode ser utilizado para auxiliar no desenvolvimento de software diretamente pelo terminal, com suporte a:

- Criação e edição de arquivos
- Navegação e exploração de repositórios
- Execução de comandos git (commits, push, pull requests)
- Geração automática de documentação
- Análise e revisão de código

## Pré-requisitos

- [Node.js](https://nodejs.org/) >= 18
- [Claude Code CLI](https://claude.ai/code) instalado globalmente:

```bash
npm install -g @anthropic-ai/claude-code
```

## Como Usar

1. Clone o repositório:

```bash
git clone https://github.com/KelvinKent/Example_ClaudeCode.git
cd Example_ClaudeCode
```

2. Inicie o Claude Code no diretório do projeto:

```bash
claude
```

3. Interaja com o Claude diretamente pelo terminal para realizar tarefas de desenvolvimento.
